# Niri + DMS Keybindings Cheat Sheet

**Mod Key:** Super (Windows Key)

---

## DMS Applications & Utilities

| Keybind         | Action                           |
| --------------- | -------------------------------- |
| `Mod + Space`   | Application Launcher (Spotlight) |
| `Mod + V`       | Clipboard Manager                |
| `Mod + M`       | Task Manager (Process List)      |
| `Mod + ,`       | Settings                         |
| `Mod + N`       | Notification Center              |
| `Mod + Y`       | Browse Wallpapers                |
| `Mod + Alt + L` | Lock Screen                      |

---

## Window Management

| Keybind           | Action                       |
| ----------------- | ---------------------------- |
| `Mod + Q`         | Close Window                 |
| `Mod + Return`    | Open Terminal (Ghostty)      |
| `Mod + E`         | Open File Manager (Nautilus) |
| `Mod + Tab`       | Toggle Overview              |
| `Mod + O`         | Toggle Overview (Alt)        |
| `Mod + Shift + /` | Show Hotkey Overlay          |
| `Mod + Shift + E` | Quit Niri                    |

---

## Window Focus (Navigation)

### Vim-Style Navigation

| Keybind   | Action             |
| --------- | ------------------ |
| `Mod + H` | Focus Column Left  |
| `Mod + L` | Focus Column Right |
| `Mod + J` | Focus Window Down  |
| `Mod + K` | Focus Window Up    |

### Arrow Keys Alternative

| Keybind   | Action             |
| --------- | ------------------ |
| `Mod + ←` | Focus Column Left  |
| `Mod + →` | Focus Column Right |
| `Mod + ↓` | Focus Window Down  |
| `Mod + ↑` | Focus Window Up    |

### Mouse Navigation

| Keybind             | Action             |
| ------------------- | ------------------ |
| `Mod + Wheel Right` | Focus Column Right |
| `Mod + Wheel Left`  | Focus Column Left  |

---

## Window Movement

| Keybind           | Action            |
| ----------------- | ----------------- |
| `Mod + Shift + H` | Move Column Left  |
| `Mod + Shift + L` | Move Column Right |
| `Mod + Shift + J` | Move Window Down  |
| `Mod + Shift + K` | Move Window Up    |

---

## Workspaces

### Switch to Workspace

| Keybind           | Action                  |
| ----------------- | ----------------------- |
| `Mod + 1-5`       | Switch to Workspace 1-5 |
| `Mod + Page Down` | Next Workspace          |
| `Mod + Page Up`   | Previous Workspace      |

### Move Window to Workspace

| Keybind                   | Action                            |
| ------------------------- | --------------------------------- |
| `Mod + Shift + 1-5`       | Move Column to Workspace 1-5      |
| `Mod + Shift + Page Down` | Move Column to Next Workspace     |
| `Mod + Shift + Page Up`   | Move Column to Previous Workspace |

---

## Monitor Management (Dual Monitor)

| Keybind                  | Action                       |
| ------------------------ | ---------------------------- |
| `Mod + Ctrl + H`         | Focus Monitor Left           |
| `Mod + Ctrl + L`         | Focus Monitor Right          |
| `Mod + Ctrl + Shift + H` | Move Column to Monitor Left  |
| `Mod + Ctrl + Shift + L` | Move Column to Monitor Right |

---

## Window Resizing & Layout

| Keybind           | Action                       |
| ----------------- | ---------------------------- |
| `Mod + R`         | Switch Preset Column Width   |
| `Mod + F`         | Maximize Column              |
| `Mod + Shift + F` | Fullscreen Window            |
| `Mod + =`         | Increase Column Width by 10% |
| `Mod + -`         | Decrease Column Width by 10% |
| `Mod + [`         | Consume/Expel Window Left    |
| `Mod + ]`         | Consume/Expel Window Right   |

**Preset Column Widths:** 25%, 33%, 50%, 66%, 75%

## Media & System Controls

### Audio (Function Keys)

| Keybind            | Action                |
| ------------------ | --------------------- |
| `F1` (Volume Up)   | Increase Volume by 3% |
| `F2` (Volume Down) | Decrease Volume by 3% |
| `F3` (Mute)        | Toggle Mute           |

### Brightness (Function Keys)

| Keybind                | Action                    |
| ---------------------- | ------------------------- |
| `F5` (Brightness Up)   | Increase Brightness by 5% |
| `F6` (Brightness Down) | Decrease Brightness by 5% |

## Quick Reference: Most Used

```
Mod + Space     → App Launcher
Mod + Return    → Terminal
Mod + Q         → Close Window
Mod + H/L       → Navigate Columns
Mod + J/K       → Navigate Windows
Mod + 1-5       → Switch Workspace
Mod + Tab       → Overview
Mod + V         → Clipboard
Mod + R         → Resize Preset
Mod + F         → Maximize
```
